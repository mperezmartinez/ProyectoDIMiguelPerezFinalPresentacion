Init de la clase que se ejecuta cada vez que la abrimos.

Inicializa todo lo necesario para el funcionamiento de la ventana, como recoger las señales con los botones ,creacion de la base de datos y como no , mostrar la ventana

Comprar(self, widget)

source code 

Se Ejecuta al Pulsar el Boton Comprar,Recoge todos los valores escritos , y los añade a la base de datos.

Recoge los valores introducidos en marca,modelo,ano,precio,motor,cambio,puertas,CV y precio de venta y los añade a la base de datos , con la id siguiente a la ultima usada, en caso de que no haya ninguna id usa la 1. Al terminar de insertar , vaciamos todos los recuadros para mas comodiad de una nueva introduccion.

:param widget: Recoge el boton
